# Simple Ass Soundboard
A simple soundboard CLI program, that plays sounds provided by the user through the speaker.

This program is intented to be used alongside a virtual audio cable to route the audio through your microphone, if intented to be used in voice calls.

## Features
- Auto-detect .mp3 and .wav files provieded in the Sounds folder
- Play audio files through your speaker

## Installation
1. Download and install [Python](https://www.python.org/downloads/)
2. Install [playsound3](https://github.com/szmikler/playsound3) via pip
   ```bash
   pip install playsound3
   ```
3. Download the [source code](https://github.com/adamsophian/Simple-Ass-Soundboard/archive/refs/heads/main.zip) and run **main.py**
